## Troubleshooting

### Cannot Login?
- Ensure you're using the correct email format
- Password is case-sensitive: `Demo@123`
- Clear browser cache and try again

### No Data Showing?
- Check if you're in the correct organization
- Verify your role has proper permissions
- Contact admin to assign permissions

### Appointment Booking Issues?
- Ensure services are configured
- Check doctor availability
- Verify department assignments
